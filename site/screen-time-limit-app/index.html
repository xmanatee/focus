Daily budgets

A Screen Time limit app for apps that need a hard stop.

Some apps should not be blocked all day. Focus Blocks lets you set a daily budget, then shields the selected apps once that time is used.

When budgets work best

Useful apps

Keep YouTube, Reddit, or social apps available briefly without letting them consume the day.

Flexible days

Use limits when a fixed schedule would be too rigid.

Combined rules

Use schedule plus budget when apps should be allowed only in a window and still capped.

Set it once, then let iOS enforce it

Focus Blocks configures Screen Time shields through Apple's APIs. Once the budget is reached, selected apps are hidden system-wide until the limit resets.

For setup help, use Focus Blocks support to check budget reset behavior, Screen Time permission, iCloud sync, and per-device app selections.
