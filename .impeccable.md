# Focus Blocks Design Context

## Users
iOS power-users fighting phone addiction or trying to do deep work. They live in the same world as Opal / One Sec / Jomo users but want something that commits them without scolding them.

## Brand Personality
**Warm. Confident. Alive.** The app is not a verdict, but it is honest — it makes the commitment felt without turning the user into the enemy of their own progress. Three words: **grounded, committed, alive**.

## Aesthetic Direction

### References
- **Headspace** — warm cream surfaces, rich saturated accents, rounded everything, generous spacing, playful confidence.
- **Duolingo** — fully-rounded buttons, vivid primary, big tappable targets, no-default feel.
- **Endel** — ambient craft, near-monochromatic palette, minimal motion.

### Anti-references
- **Freedom / clinical-corporate** — no flat blue, no enterprise chrome.
- **Apple Screen Time / Settings native** — no red-circle graphs, no default rounded-blue-icon grid.

### System theme (auto dark)
App responds to iOS `Appearance` — if the user's device is in Auto Dark Mode (Settings → Display & Brightness → Automatic), the app turns dark at sunset with the rest of the system. No in-app toggle; we follow the OS.

Light and dark palettes are defined as CSS variables in `global.css` and consumed via NativeWind's `darkMode: 'class'`.

### Typography
SF Pro only. Weight contrast is the signature: `display-md` Black for headlines, `label` short-caps tracked, `body-md` 17pt semibold (Duolingo-weighted body), `caption` 14pt.

### Roundness
Every surface reads rounded:
- Buttons → `rounded-full` (pill). Duolingo-level commitment.
- Inputs → `rounded-xl` (24px).
- Preset chips / day chips → `rounded-full`.
- Cards / rails → `rounded-2xl` (32px).
- Icons → native SF Symbols (geometric consistency).

### Motion
Reanimated, exponential ease-out (no bounce). Press: `scale 0.97 @ 140ms`.

### Haptics
- `commit` — Rigid (grant, save)
- `select` — selectionAsync (chip change, toggle)
- `abandon` — Soft impact (user aborts) — **never Error**

### Iconography
SF Symbols via `expo-symbols`. No emoji. No custom PNG icons.

## Design Principles
1. **Warm over cool.** Every neutral is warm-tinted. Pure black and white never appear.
2. **One accent, scarce.** Signal orange appears only on commit / active / primary-action surfaces.
3. **Rounded everything.** Pills for action, 2xl for surfaces. No sharp corners in the main flow.
4. **Typography carries hierarchy.** Black + semibold, not color tiers.
5. **No guilt design.** Abandonment is soft. No red, no scolding, no "you failed" copy.
6. **System theme wins.** Focus Blocks never decides for the user; iOS does.
